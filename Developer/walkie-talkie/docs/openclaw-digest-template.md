# Openclaw Daily Digest Template (TASK-D2)

This document outlines a proposed template for the Openclaw daily digest. The goal of this digest is to provide a concise, at-a-glance overview of the team's activity, allowing a personal assistant to quickly get up to speed.

## 1. Digest Structure

The digest will be broken down into the following sections:

*   **`Yesterday's Highlights`**: A brief summary of the most important accomplishments from the previous day.
*   **`Today's Priorities`**: A list of the top 3-5 priorities for the current day.
*   **`Key Decisions`**: A summary of any key decisions made in the last 24 hours.
*   **`Blockers`**: A list of any blockers that are impeding progress.
*   **`Agent Status`**: A table showing the status of each agent on the team.

## 3. Automation

This digest can be automatically generated by a dedicated agent at the beginning of each day. The agent would be responsible for gathering the necessary information from our shared Obsidian vault and other sources, and then compiling it into the digest format.
